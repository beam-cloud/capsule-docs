Capsule turns one Python app definition into a hosted AI product.

You write the app. Capsule runs the runtime, chat surface, hosted UI, auth, state, background work, integrations, and deploy loop around it.

## fast setup

Install the SDK and CLI:

```bash
uv add cpsl
```

or:

```bash
pip install cpsl
```

Authenticate once:

```bash
capsule login
```

Create `app.py`:

```python
import cpsl
from openai import AsyncOpenAI

client = AsyncOpenAI()

app = cpsl.App(
    name="hello-capsule",
    image=cpsl.Image(python_packages=["openai"]),
    secrets=["OPENAI_API_KEY"],
)


@app.page("Status", icon="layout-dashboard")
def status_page():
    return cpsl.ui.Page(
        [
            cpsl.ui.Text("Hello from Capsule", style="heading"),
            cpsl.ui.Text("Chat and pages come from the same app definition.", style="muted"),
        ]
    )


@app.message()
async def handle(session: cpsl.Session, msg: cpsl.Message):
    response = await client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[
            {"role": "system", "content": "You are a concise operations assistant."},
            *session.chat_messages(msg),
        ],
    )
    await session.reply(response.choices[0].message.content or "No response.")
```

Run it:

```bash
capsule serve app:app
```

## prod setup

When the local app works, deploy the same entry point:

```bash
capsule deploy app:app
```

The deployed app gets a hosted URL, app-scoped sign-in, and the same registered chat handler, pages, collections, tasks, integrations, channels, secrets, and filesystems declared in Python.
